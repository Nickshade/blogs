diff 命令

作用

    比较文件差异(用于行比较)

参数

    -u:输出带有上下文

例子

    diff -u <file1> <file2>
    diff -u <file1> <file2> > diff.txt

patch命令

作用

    apply a diff file to an original

例子

    patch <file2> < diff.txt
    patch -R <file1> < diff.txt

git命令

  • 查看git版本:

    git –version
