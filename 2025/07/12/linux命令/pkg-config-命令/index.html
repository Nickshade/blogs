pkg-config 命令

YAOHAN-LI
2025-07-12

pkg-config 是 Linux 下的一个用于管理库依赖信息的工具，常用于获取编译和链接某个库所需的参数（如 -I, -L, -l 等）。

它是开发 GTK、OpenSSL、libcurl、GStreamer 等库项目时的重要工具。

✅ 一、作用总结

  • ✅ 获取头文件路径（-I）
  • ✅ 获取链接库参数（-L, -l）
  • ✅ 获取版本信息
  • ✅ 检查库是否已安装

✅ 二、常用命令选项

命令 含义
pkg-config --cflags <lib> 输出编译时需要的头文件路径（-I...）
pkg-config --libs <lib> 输出链接时需要的库参数（-L... -l...）
pkg-config --libs --cflags <lib> 同时输出编译和链接参数
pkg-config --modversion <lib> 显示库的版本号
pkg-config --exists <lib> 判断库是否存在（不输出内容，用于 shell 判断）
pkg-config --list-all 列出所有可用的库及描述
PKG_CONFIG_PATH=... pkg-config ... 指定额外的 .pc 文件搜索路径

✅ 四、.pc 文件原理简介

pkg-config 实际上是读取 .pc 文件，这些文件通常位于以下路径之一：

  • /usr/lib/pkgconfig/
  • /usr/share/pkgconfig/
  • /usr/local/lib/pkgconfig/
  • 用户自定义：可通过 PKG_CONFIG_PATH 设置
